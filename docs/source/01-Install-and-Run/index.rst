安装与运行
==============================================================================


运行
------------------------------------------------------------------------------

- 此游戏只能在 Windows 平台上运行.
- Dos 版本需要借助 DoxBox 运行.
- 如果你在 Windows 系统上运行时出现乱码, 请进入控制面板, 设置, Region, Admin, 选择 ``使用下面的编码打开应用程序`` 为 China (Simplify).
