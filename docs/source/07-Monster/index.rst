怪物资料一览
==============================================================================

屬性相克：風克雷、雷克水、水克火、火克土、土克風

怪物/敵人/魔物對劍系法術的抗性為50%。

怪物身上的道具能夠全部偷到；怪物身上的錢要分2-3次才能偷完。

靈葫值等於 ``0`` 的怪物不能被紫金葫蘆吸入；靈葫值不等於 ``0`` 的怪物則可以被吸入，被吸入的怪物的靈葫值會在紫金葫蘆內累積，作為煉丹素材。纍積靈葫值和煉成品的關係為： 行軍丹1，還神丹2，還魂香3，試煉果4，舍利子5，蜂王蜜6，孟婆湯7，蟠果8，靈葫仙丹9。
