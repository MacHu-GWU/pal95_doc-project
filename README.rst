Install
------------------------------------------------------------------------------

``pal95_doc`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install pal95_doc

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade pal95_doc
